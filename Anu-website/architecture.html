Hardware-Software Connection

Hardware Components

Raspberry Pi 5

Main processing unit - connects all hardware

GPIO Pins, USB, I2C, SPI

PCA9685 Servo Driver

Controls 17 servo motors via I2C

Address: 0x40, 16 channels

Sensors

GPIO-connected sensors

Ultrasonic: GPIO 23/24, PIR: GPIO 25/26

USB Devices

Camera and microphone

USB 3.0 interface
